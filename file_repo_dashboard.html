# Widget Type Calculation
1 Total Files KPI COUNT(*) from file_metadata
2 Overdue for Disposal KPI COUNT(*) WHERE disposal date < NOW()
3 First Analysis PASSED Stat card this week: first_analysis_result = true AND first_analysis_complete_at ≥ DATE_TRUNC('week', NOW())
this year: first_analysis_result = true AND first_analysis_complete_at ≥ DATE_TRUNC('year', NOW())
4 First Analysis FAILED Stat card this week: first_analysis_result = false AND first_analysis_complete_at ≥ DATE_TRUNC('week', NOW())
this year: first_analysis_result = false AND first_analysis_complete_at ≥ DATE_TRUNC('year', NOW())
5 Files Received This Hour KPI COUNT(*) WHERE file_received_at ≥ NOW() - INTERVAL '1 hour'
6 Files Received This Week KPI COUNT(*) WHERE file_received_at ≥ DATE_TRUNC('week', NOW())
7 Files Received This Year KPI COUNT(*) WHERE file_received_at ≥ DATE_TRUNC('year', NOW())
8 Processing Lag — Avg & P95 per Stage Stat cardfull width received→1st analysis: AVG / MIN / MAX / P95 of (first_analysis_complete_at - file_received_at)
received→2nd analysis: AVG / MIN / MAX / P95 of (second_analysis_complete_at - file_received_at)
9 File Category Spread Pie chart COUNT(*) GROUP BY file_category
10 Files by Customer Type Pie chart COUNT(*) GROUP BY customer_type

12 Files by Creator Horiz. bar COUNT(*) GROUP BY file_creator ORDER BY count ASC
